# HunTianDB (混天DB)

HunTianDB is a high-performance timeseries security database with full PostgreSQL Wire Protocol v3.0 compatibility. Built in Rust for speed, safety, and reliability. Designed for security audit trails, financial event streams, and real-time observability workloads.

## Key Features

- **PostgreSQL Wire Protocol v3.0:** Zero-friction compatibility with psql, DBeaver, pgAdmin, JDBC, psycopg2, and any standard PostgreSQL client. No migration cost.
- **Async Lock-Free WAL:** Write path uses crossbeam channels with a background writer thread. Client returns instantly; WAL is persisted asynchronously with zstd-compressed bincode format (5x smaller than JSON).
- **Crash-Safe Design:** CRC32 WAL checksums, LSN checkpoint recovery, and synchronous commit modes ensure zero data loss even on kill -9.
- **Vectorized Aggregation:** Columnar cache with contiguous f64 slices. COUNT(*) in 0.07ms on 100K rows — 500x faster than PostgreSQL 16.
- **SCRAM-SHA-256 Authentication:** Production-grade password verification with salted hashes. Built-in admin, writer, and reader roles.
- **Docker Single Binary:** One image, three ports — PG wire (5408), REST API + Web Portal (3000), Prometheus metrics (5490).
- **Prometheus Observability:** Built-in metrics endpoint with histograms, gauges, and counters for WAL latency, memory usage, query duration, and more.
- **Multi-Language Support:** Client examples in Python, Go, Java/JDBC, Node.js, Rust, C/libpq, Erlang, Haskell, PHP, Ruby, and psql.

# 混天DB (HunTianDB)

混天DB（HunTianDB）是一款高性能时序安全数据库，完整兼容 PostgreSQL Wire Protocol v3.0。采用 Rust 语言打造，追求极致的速度、安全与可靠性。专为安全审计追踪、金融事件流和实时可观测性场景而设计。

## 项目信息

- **名称：** 混天DB (HunTianDB)
- **类型：** 时序安全数据库
- **开发语言：** Rust
- **许可证：** MIT
- **最新版本：** v0.1.7（2026-05-20）
- **代码仓库（国际）：** https://github.com/ctkqiang/HunTianDB
- **代码仓库（中国）：** https://gitcode.com/ctkqiang/HunTianDB
- **Docker Hub：** ctkqiang/huntianandb
- **阿里云容器镜像：** crpi-onofuhwrkmb5z0mn.cn-hangzhou.personal.cr.aliyuncs.com/nezhawanluoanquan/huntiandb

## 核心特性

- **PostgreSQL Wire Protocol v3.0：** 原生 PG 线协议。无缝兼容 psql、DBeaver、pgAdmin、JDBC、psycopg2 及所有标准 PostgreSQL 客户端，零迁移成本。
- **异步无锁 WAL：** 写入路径基于 crossbeam 无锁通道，客户端即刻返回，后台线程异步处理 zstd 压缩与磁盘同步。Bincode 格式比 JSON 小 5 倍。
- **崩溃安全设计：** CRC32 WAL 校验和、LSN 检查点恢复、同步提交模式。即使 kill -9 强制终止，重启后数据零丢失。
- **向量化聚合：** 列式缓存配合连续 f64 切片，直接迭代。10 万行 COUNT(*) 仅需 0.07ms，比 PostgreSQL 16 快 500 倍。
- **SCRAM-SHA-256 认证：** 生产级密码验证，加盐哈希存储。内置管理员（admin）、写入者（writer）、只读（reader）三种角色。
- **Docker 单镜像部署：** 一个镜像，三个端口。PG 线协议（5408）、REST API + Web Portal（3000）、Prometheus 指标（5490），一键拉取即用。
- **Prometheus 可观测性：** 内置指标端点，提供直方图、仪表、计数器，覆盖 WAL 延迟、内存占用、查询耗时等关键指标。
- **多语言客户端：** 提供 Python、Go、Java/JDBC、Node.js、Rust、C/libpq、Erlang、Haskell、PHP、Ruby、psql 共 11 种语言的连接示例。
